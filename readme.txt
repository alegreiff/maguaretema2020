Theme maguare WP

== Descripción ==

Theme WP para Maguaré http://maguare.gov.co/ : Portal infantil de la Estrategia Digital de Cultura y Primera Infancia del Ministerio de Cultura de Colombia

== Instalación ==
	
1. En el panel de Administración, ir a Apariencia > Temas > Agregar nuevo.
2. Click en Cargar Tema y seleccionar el Tema maguare, al seleccionar el archivo .zip del Theme. Instalar Ahora.
3. Click en Activar y usar sus opciones.

== Preguntas Frecuentes ==

= Este tema que Plugins Soporta? =

Maguaré usa los siguientes Plugins:

WP Multifunctional - Plugin que contiene varias funciones para WP , optimizado para performance y seguridad.
advanced-custom-fields-pro - Plugin Que crea Campos Personalizados para todo el contenido
contact-form-7 - Plugin Formularios de Contacto
real3d-flipbook -Plugin para visualizar libros en PDF-JPG
better-wp-security - Plugin de Seguridad WP
wordpress-seo - Plugin de SEO
wp-extended-search - Plugin para extender la búsqueda en WP
ajax-search-pro - Plugin para Buscador Avanzado
contact-form-7-mailchimp-extension - Plugin Suscripción MailChimp Maguaré
